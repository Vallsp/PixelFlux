# Introduction

Welcome to the Pixelflux handbook. This book gathers everything about the
project in one place: how to run it, how it is built, how it is tested and
secured, and how it deploys. Every page is generated from the canonical
Markdown in the repository, so it never drifts from the source.

📊 Prefer the short version? The
[project slides](https://vallsp.github.io/PixelFlux/slides/) — a Marp deck on the
architecture and SDLC — are authored as Markdown and published right beside this
book.
